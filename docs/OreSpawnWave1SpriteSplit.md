# OreSpawn Wave 1 sprite split

This file now tracks the updated art direction for the first OreSpawn prototype wave.

## Mob plan

All Wave 1 mobs are wired with clean placeholder spritesheets for gameplay testing first.

- `Water Dragon`
- `Mantis`
- `Emperor Scorpion`
- `Hercules`
- `Caterkiller`
- `Cephadrome`

These sheets are intentionally temporary so we can validate scale, hitbox, frame count, AI, spawn conditions, and drops while you create the final monster art.

## Item plan

The item side is where the visual pass is moving forward already.

- Materials, summons, and `Big Hammer` now use custom icons with stronger silhouettes and direct inspiration from the OreSpawn wiki item art.
- Mob sprites stay neutral placeholders until your final sheets replace them.
