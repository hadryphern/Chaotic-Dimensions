# OreSpawn Master Content Codes

This document maps every gameplay-facing OreSpawn page I could discover on the wiki mirror into a proposed internal code key for `Chaotic Dimensions`.

Scope note:
- This is exhaustive relative to the discoverable gameplay pages on the mirror as crawled on `2026-03-26`.
- When a source page is a gallery or family page, the code key below is a bundle/registry key, not a single Terraria item or NPC.
- Event keys are partly inferred from page behavior, because the wiki often describes systems and gameplay loops rather than naming them as formal events.

## Prefix Legend

- `OreBoss.*` = official boss-scale encounter.
- `OreNpc.*` = single NPC / mob / companion page.
- `OreNpcGroup.*` = one wiki page covers multiple variants or stages.
- `OreItem.*` = singular item, weapon, tool, or utility page.
- `OreSet.*` = armor tier or tools-and-weapons family page.
- `OreRegistry.*` = gallery/collection page that should later be split into many concrete content entries.
- `OreDim.*` = dimension/world layer.
- `OreEvent.*` = inferred Terraria-side event/system key.
