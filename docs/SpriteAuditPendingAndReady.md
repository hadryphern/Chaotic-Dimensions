# Chaotic Dimensions Sprite Audit

Atualizado em `2026-03-28`.

Este arquivo é a auditoria completa de sprites do mod, com foco no que ainda precisa de arte final.

## Legenda

- `✅` já tem sprite própria e hoje está em um estado bom/usável
- `⚠️` já existe arquivo `.png`, mas a arte atual parece placeholder, protótipo ou base temporária
- `❌` ainda não tem sprite exclusiva; hoje usa arte vanilla, arte compartilhada ou desenho por código

## Regras rápidas

- `Total recomendado` = tamanho total da spritesheet
- `Frame recomendado` = tamanho de cada frame
- `Frames` = quantidade sugerida
- layout padrão recomendado para mobs e bosses: `vertical`

## Observações finais

- Para armaduras equipáveis, siga exatamente os nomes:
- `_Head`
- `_Body`
- `_Arms`
- `_FemaleBody`
- `_Legs`

- Para whips, o projectile é uma tira vertical única com:
- cabo
- segmento
- ponta

- Para minérios do mundo:
- item do inventário continua separado
- tile do mundo de minério estilo Terraria deve usar `288x270` quando for terrain sheet completa

- caminho
